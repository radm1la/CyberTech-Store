# CyberTech Store

A futuristic cyberpunk-themed e-commerce platform for tech products featuring an immersive terminal-inspired interface.

## Live Demo

https://cyber-tech-store.vercel.app

## Overview

CyberTech Store is a modern e-commerce web application that combines cutting-edge web technologies with a unique cyberpunk aesthetic. The platform delivers an immersive shopping experience through a terminal-style interface, advanced product filtering, and secure user authentication.

## Features

- Cyberpunk-inspired UI with terminal aesthetics
- User authentication system
- User profile management:
  - Edit personal information
  - Change password
  - Delete account
- Advanced product filtering:
  - Category selection
  - Brand filtering
  - Price range slider
  - Star rating filter
- Dynamic product database
- Responsive design for all devices
- Fast performance and smooth interactions
- Terminal-style filter interface

## Tech Stack

- **Frontend**: JavaScript, HTML5, CSS3
- **API**: EverREST API (https://api.everrest.educata.dev)
- **Authentication**: JWT (JSON Web Tokens)
- **Deployment**: Vercel
- **Architecture**: Vanilla JavaScript (no frameworks)

## API Integration

This project uses the **EverREST API** for backend functionality:

- User authentication (sign up, sign in, sign out)
- User profile management (edit profile, change password, delete account)
- Product catalog with search and filtering
- Category and brand filtering
- Price range and rating filters

API Base URL: https://api.everrest.educata.dev
